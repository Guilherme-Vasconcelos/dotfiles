# My Dotfiles

Configuration files for bash (.bashrc and .aliases) and Noevim (init.vim)

## Installation

```bash
cd $HOME
git clone https://github.com/johnvictorfs/dotfiles.git
cd dotfiles
chmod a+x install.sh && ./install.sh
```

### Change Plank theme

```bash
plank --preferences
```
